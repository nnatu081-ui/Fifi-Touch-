# Habesha Near Me — MVP

A mobile-first community marketplace prototype for discovering Habesha businesses, services, store deals, jobs and events by city.

## Included
- City selection and nearest-city detection
- Search and category filters
- Business/service, store deal, event and job cards
- WhatsApp contact flow
- English/Tigrinya interface toggle
- Submission form saved in localStorage
- Share buttons

## Current MVP limitation
Submissions are stored on the visitor's device and sent for approval through WhatsApp. A production version should connect the form to Supabase, add authentication, image uploads and an admin moderation dashboard.
